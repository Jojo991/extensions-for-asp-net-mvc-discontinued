##### **Project Description** #####
This is an example project that I put together for my blog post: EditorTemplates, DataAnnotations and Telerik, Oh my. located at http://nederveld.wordpress.com/2011/02/16/editortemplates-dataannotations-and-telerik-oh-my/

It shows how you can use the Telerik MVC Editor with Data Annotations and EditorTemplates.
