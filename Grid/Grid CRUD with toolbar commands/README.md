|Programming language               |C#|
|Browser                            |all browsers supported by Telerik Extensions for ASP.NET MVC|

##### **Project Description** #####
This project shows how to use custom toolbar commands and the grid client-side API in order to implement CRUD (Create, Read, Update, Delete) without the need of a command column.
